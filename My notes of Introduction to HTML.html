INTRODUCTION TO HTML

What is meant by HTML?

HTML is a abbreivation of "Hyper Text Markup Language". It is a standard markup language for creating web pages. It describes the structure of a web page. It consists of series of elements and tells the browser how to display the content. HTML elements consists of a pieces of contents such as "This is Heading", "This is paragraph", "This is body" like so.

The HTML element is defined by start tag, some content and end tag. Generally, web browsers are used to read the html pages and display them correctly
